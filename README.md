# CurrentAffairs-2022
A Project to Test our Knowledge on Current Affairs in 2022. This Project is created by using 'MYSQL', 'JAVA SWING', 'HASHMAP', 'RECURRSION', and 'STRINGS'. 
It is created in NetBeans.
